

# Lokesh Cinematic Universe (LCU) – Official Website

A cinematic, dark-themed, Marvel-inspired website showcasing the connected Tamil action universe created by Lokesh Kanagaraj.

---

## 1. Design System & Theme
- **Dark cinematic palette**: Black/charcoal background, deep red (#8B0000) and gold (#D4AF37) accents
- **Bold cinematic typography**: Large impactful headings, clean body text
- **High contrast cards** with sharp edges and glow effects
- **Consistent glow buttons** with red/gold hover animations

## 2. Loading Screen & Page Intro
- Cinematic logo reveal animation on first load
- "LCU" text with golden glow fade-in effect
- Smooth transition into the main site

## 3. Sticky Navigation Bar
- Transparent navbar that becomes solid on scroll
- Smooth scroll links: Home, Movies, Timeline, Characters, About
- LCU logo on the left, navigation links on the right
- Mobile hamburger menu with slide-in drawer

## 4. Hero Section (Full-Screen Landing)
- Full-screen static cinematic background with dark gradient overlay
- Particle/light flare animations for atmosphere
- Large animated title: "LOKESH CINEMATIC UNIVERSE"
- Subtext: "A Connected Tamil Action Universe"
- Scroll-down indicator with bounce animation

## 5. Movies Section (Horizontal Scroll)
- Horizontally scrollable movie cards (swipeable on mobile)
- **4 movies**: Kaithi (2019), Vikram (2022), Leo (2023), Rolex (Upcoming)
- Each card: Styled placeholder poster, title, year, "Explore" button
- Hover effects: scale up + glow border
- Clicking "Explore" → opens dedicated movie detail page with:
  - Full-width cinematic banner
  - Synopsis section
  - Cast grid
  - Timeline placement callout
  - Embedded YouTube trailer (placeholder link)
  - Image gallery carousel

## 6. LCU Timeline Section
- Interactive animated timeline: Kaithi → Vikram → Leo → Future
- Glowing connection lines between nodes
- Scroll-triggered reveal animations
- Each node shows movie title, year, and brief description

## 7. Characters Section
- Grid of cinematic character portrait cards
- Characters: Dilli, Vikram, Leo Das/Parthiban, Rolex
- Hover effect: image darkens, name + short bio fades in
- Styled placeholder portraits with character colors

## 8. About Lokesh Kanagaraj Section
- Director spotlight with placeholder portrait
- Short biography
- Vision statement for the LCU
- Cinematic style explanation
- Parallax background effect

## 9. Animation & Effects (Throughout)
- Scroll-triggered fade-in and slide-up animations on all sections
- Parallax scrolling on backgrounds
- Custom cursor with subtle glow trail
- Cinematic hover effects on all interactive elements
- Smooth page transitions between routes
- Glow pulse effects on accent elements

## 10. Footer
- Social media icon links
- Copyright notice
- "LCU is a conceptual universe created by Lokesh Kanagaraj"
- Dark themed, minimal design

## 11. Responsiveness & Performance
- Fully responsive across mobile, tablet, and desktop
- Horizontal scroll converts to swipe on mobile
- Lazy loading for images
- SEO meta tags on all pages
- Optimized for fast performance
