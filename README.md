# LCU - Lokesh Cinematic Universe 🎬

A modern, interactive website showcasing the Lokesh Cinematic Universe - featuring in-depth details about movies, cast information, and the interconnected storylines of director Lokesh Kanagaraj's cinematic universe.

## 🎯 About

The **Lokesh Cinematic Universe (LCU)** is a comprehensive web application dedicated to exploring the films and characters within director Lokesh Kanagaraj's interconnected cinematic universe. This project provides an immersive experience for fans to discover movie details, cast information, character arcs, and the narrative connections between films.

## ✨ Features

- 🎥 **Movie Showcase** - Browse and explore all Lokesh movies with detailed information
- 👥 **Cast & Crew** - Comprehensive cast information and actor profiles
- 📖 **Movie Details** - Plot summaries, release dates, ratings, and reviews
- 🔗 **Universe Connections** - Discover how films connect within the LCU
- 🎨 **Modern UI** - Responsive and elegant user interface
- ⚡ **Fast & Performant** - Optimized for speed and smooth interactions
- 📱 **Mobile Responsive** - Seamless experience across all devices

## 🛠️ Tech Stack

- **Frontend Framework**: TypeScript (97.5%)
- **Styling**: CSS (1.6%)
- **Architecture**: Modern component-based structure

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14 or higher)
- npm or yarn package manager

## 🚀 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Varunmattur/LCU.git
   cd LCU
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open your browser**
   - Navigate to `http://localhost:3000` (or the configured port)
